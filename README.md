## Clone this repo

###### Notes: Check the .env.example before clone this repo

```
git clone git@github.com:fandredev/boiterplate-nestjs-youtube.git
```

## Go to directory

```
 cd boiterplate-nestjs-youtube
```

## Install dependencies

```
  yarn
```

## Run server

```
 yarn run start:dev
```
